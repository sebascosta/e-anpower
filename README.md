Installation

Install the dependencies and devDependencies and start the server. ---Follow the instructions---

1.- git clone https://github.com/sebascosta/e-anpower.git

2.- cd Anpower\e-anpower

3.- npm install | yarn

4.- npm run dev | yarn dev

5.- Ready to edit!
